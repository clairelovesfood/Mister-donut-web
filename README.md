Mister Donut UI Project

A responsive HTML, CSS, and JavaScript project built to recreate and enhance a Mister Donut interface.

Features:
	•	Responsive layout optimized for desktop, tablet, and mobile viewports
	•	Clean styling with improved visual hierarchy
	•	Interactive buttons with JavaScript functionality
	•	Accessibility-friendly structure
	•	Modern CSS layout techniques (flexbox / media queries)

Tech Stack
	•	HTML5
	•	CSS3
	•	JavaScript

Goals:

This project focuses on:
	•	Responsive web design
	•	UI polish and visual consistency
	•	Functional interactivity
	•	Cross-device usability
